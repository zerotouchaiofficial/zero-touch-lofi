This guide will help you set up and start using Zero Touch Music.

## Requirements

- Python 3.9+
- YouTube Data API credentials
- SMTP-enabled email account
- FFmpeg installed

## Installation

```bash
git clone https://github.com/zerotouchaiofficial/Zero-Touch-Music.git
cd Zero-Touch-Music
python -m venv venv
source venv/bin/activate  # macOS/Linux
venv\Scripts\activate     # Windows
pip install -r requirements.txt
```

## Run the uploader

```bash
python main.py
```
