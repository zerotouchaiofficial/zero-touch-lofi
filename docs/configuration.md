# ⚙️ Configuration Options

Zero Touch Music can be customized for video, metadata, scheduling, and email alerts.

## Video

- Resolution
- Frame rate
- Visualizer style
- Background animations

## Metadata

- Title
- Description
- Tags
- Thumbnail

## Scheduler

- Upload intervals
- Publish time
- Timezone

## Email Alerts

- Sender/recipient credentials
- Success/failure notifications
